# PyCloud - simple file-cloud service
esta es una aplicacion de almacenamiento en nube tipo SaaS parecida a servicios
como google drive o dropbox. Este proyecto es solo experimental y con fines
academicos, por lo que no contendra al 100% todas las caracteristicas que
ofrecen los servicios antes mencionados pero que a futuro se pretenden
implementar.

## Powered by Django
este proyecto esta realizado bajo el framwork de django de python principalmente
aunque tambien se utilizan tecnologias web como lo son HTML, CSS y JS.

## Dependencias
Las dependencias principales para el proyecto son:
- Python 3.12.x a 3.14.x
- Django (con pip) 6.0 o superior
- Nginx (despliegue) cualquier version, de preferencia reciente.

Nota: para las tecnologias web no es necesario instalar nada, con un navegador
es mas que suficiente.
